Replace these files in your project:
- app.py
- schema.sql
- requirements.txt

Render:
- Environment variable DATABASE_URL must already be set to your Neon connection string
- Start command: gunicorn app:app

Important:
- Database data will be stored in Neon and remain permanent.
- Local uploaded files in the uploads folder are still stored on Render's local disk, so uploaded files may still disappear on Render Free restarts.
